Project Management Application
Project Management Application – example of Angular client Single Page Application (SPA) with JSON Web Token Authentication, REST API Server and MySQL database Server
The Application was developed using
  • Angular 8 platform and framework
  • Angular Material 8 - Material Design components for Angular
  • NodeJS  and Angular CLI
  • MySQL 5.7
  • HTML 5 and CSS 3, Style preprocessors:  SCSS, SASS, LESS
  • TypeScript and ECMAScript 2015 (ES6 JavaScript)
  • RxJS 6 (Reactive Extensions Library for JavaScript)

The REST API Server implement JWT Authentication in a NodeJS API (Express).
The Application Backend  MySQL SQL Database has been implemebted with Stored Procedures.

Used Angular buildings blocks:
  • NgModules, Components, Directives, Pipes
  • Services and Dependency Injection (DI)
  • Reactive Forms and Forms Validations
  • RxJS Observables and Subjects
  • HttpClient - client HTTP API to communicate with backend services, Interceptors
  • Router and Guards
The Application functions:
  • signup (create) User
  • login User by user name and email
  • create, read, update, delete (CRUD operations) on one or many Projects
  • in selected Project create, read, update, delete (CRUD operations) on one or many Tasks

Data model:
User (1 : M) Projects (1 : M) Tasks
